# 设备连接数量问题

目前所有电信天翼宽带，都是统一被限制只允许一台插网线使用的设备和一台无线设备同时使用，无线设备包含（笔记本的无线网卡、手机、平板、无线网卡产品等），此上述设备都是无线网的定义范围内的。要是两台无线设备同时登陆，会出现两种情况，一是账号会被封号，二是前者账号会被踢下线

# 登陆失败？

没有订购此产品：登录不上宽带的，先确认订单是否归档（广州电信小程序查看），归档或受理成功待竣工后前往广东电信小程序进行宽带充值，之后便可以使用了（具体操作群文件有）
用户认证失败：宽带账户那（即学号) 后面别加 @GCP；别直接收藏电信验证链接，IP 变化会用户验证失败
* 新开通宽带需先氪 30 块进去，不然等第二天账户被暂停就别找我

# 掉线？

YOUNG / 有线（即校园网）的验证是 24 小时有效认证，每次重新链接或者 24 小时之后需要重新进行上网认证
如有出现登陆成功但没有网，原因都是出现了 “伪登陆” 导致，目前都不建议把电信天翼网登陆界面进行收藏，建议收藏以 http 协议为准的其它网站进行跳转，这里需要另外说明，https 协议网站是无法跳转弹出登陆界面，因为认证服务器不接受 https 这个协议的访问。
宽带费用统一每月 30 元（50M 卡话费不能代替扣费）

20M/50M 单独广东电信小程序进行宽带缴费

链接网络有问题的请联系群主寻求帮助.

# 校园网报修 MAC 地址绑定错误

广州城市职业学院网上运营厅 (http://rwsx.gcp.edu.cn/)


请自己选择报修内容
网口坏了：端口或线路问题
GCPWIFI：无线 MAC 地址绑定错误绑定
锐捷提示 MAC 地址绑定错误：有线 MAC 地址绑定错误

此次解绑，需要为设备正确填写 IP 地址信息、安装认证软件后，方能进行认证，认证后会绑定初次认证设备，并且只能绑定这个设备上网，若要更改认证设备，请自行网络报障，按格式填写报障单，然后会有专人处理；现解绑的设备是用于连接宿舍有线网络连接用；若要解绑公共 wifi 连接，请重新报障注明解绑无线 MAC 地址

订单提交超时属于正常情况，提交完即可
