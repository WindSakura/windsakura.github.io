# 校园网有线配置

首先提示，内网和校园网的密码各不相同 内网是八位数出生日 校园网的是你自己设置

还有，可以直接连 YOUNG 进行申请，不用问我怎么进 VPN，VPN 密码？= 门户密码

# 一、学生入网申请

  1. 链接 YOUNG 或者学校 VPN 打开广州市城市职业学院校园服务厅网站 （http://rwsx.gcp.edu.cn）
  2. 登录服务厅。进入校园服务厅页面后进行用户登录，账号为学号，密码为八位数出生日（登录后可自行修改密码）；
  3. 自助办理入网申请。登录后，点击 “业务办理入网申请” 进行入网业务申请，按要求进行填写 “入网申请表”（注：所有资料必须认真正确填写，否则将无法开通校园网），填写资料后上传身份证图片（正面及背面，单个文件不超过 2MB，建议微信拍照保存）然后点击 “提交” 即可。（端口和 MAC 无须填写）
  4. 等待审批消息。办理成功后，在 “业务办理” 点击查看入网申请项目的 “办理跟踪” 进行查询结果，审批通过后，入网状态即为 “已激活”；同时入网信息也会同步推送到入网申请时填写的邮箱。
  5. 配置电脑上网 IP 信息。在收到入网成功的推送消息后，按照所分配 IP 信息，配置电脑
  6. 安装电脑上网认证客户端软件。安装认证客户端，输入所分配的账号和密码后 **（即学号和八位数生日）**，登录认证，即可上内网。

如有出现登陆成功但没有网，原因都是出现了 “伪登陆” 导致，目前都不建议把电信天翼网登陆界面进行收藏，建议收藏以 http 协议为准的其它网站进行跳转，这里需要另外说明，https 协议网站是无法跳转弹出登陆界面，因为认证服务器不接受 https 这个协议的访问。

# 进行外网上网认证

  • 先关闭 Wi-Fi，然后打开任意浏览器

  • 键入 www.baidu.com 或 http://hao.360.com 等具有 http 链接的网址

  • 然后他会劫持到电信的上网认证页面

    然后输入学号（后面请勿加入 @GCP），密码，进行认证，登陆成功即可上网冲浪
